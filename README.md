# Forge Docs — Lingtarn Forge 官方文档

本仓库为 [Lingtarn Forge](https://github.com/yinnxinn/forge-docs)（灵碳云铸）的**官方文档**，采用 GitBook 结构，支持中英双语。

## 文档入口

- **英文**：根目录 [gitbook/](gitbook/README.md)，[目录](gitbook/SUMMARY.md)
- **中文**：[gitbook/zh/](gitbook/zh/README.md)，[目录](gitbook/zh/SUMMARY.md)

## 本地构建

```bash
# 使用 HonKit（GitBook 分支）
npx honkit build

# 或 GitBook CLI
npx gitbook-cli build . ./_book
```
